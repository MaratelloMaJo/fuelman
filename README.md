# FuelMan

FuelMan — это мобильное приложение для умного учёта расхода топлива и электроэнергии вашего автомобиля.

## Основные возможности

* **Учёт заправок и зарядок:** Поддержка автомобилей с двигателями внутреннего сгорания (бензин, дизель), гибридов (PHEV, HEV, MHEV) и полностью электрических авто (BEV).
* **Алгоритм Full-to-Full:** Автоматический расчёт среднего расхода топлива при условии заправки "полного бака".
* **Статистика:** Детальная аналитика затрат, расхода топлива и пробега, визуализированная с помощью графиков.
* **Поддержка нескольких автомобилей:** Удобное управление гаражом с переключением между разными транспортными средствами.
* **Гибкие настройки:** Поддержка различных единиц измерения (Литры, Галлоны, кВт⋅ч, км, мили) и валют (RUB, KZT, USD, EUR).
* **Экспорт данных:** Возможность экспорта истории заправок в формат CSV для последующего анализа в Excel.
* **Резервное копирование:** Импорт и экспорт базы данных для сохранения ваших записей.
* **Напоминания:** Настраиваемые уведомления, если вы давно не вносили данные о заправке.
* **Темная тема:** Поддержка светлой и тёмной тем оформления.

## Используемые технологии

* **Flutter:** Кроссплатформенный фреймворк для разработки пользовательского интерфейса.
* **GetX:** Управление состоянием приложения и навигация.
* **SQLite (sqflite):** Локальная база данных для хранения информации о транспортных средствах и записях.
* **fl_chart:** Построение графиков и диаграмм.
* **intl:** Работа с датами и интернационализация (локализация на русский, английский и казахский языки).
* **shared_preferences:** Сохранение настроек пользователя.
